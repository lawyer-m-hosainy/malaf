# 🚀 دليل تسليم منصة مَلَف | MALAF Handover Guide

تهانينا! لقد تم إعداد منصة **مَلَف (Malaf)** لتكون جاهزة تماماً للعرض أمام المستثمرين وللإطلاق التجاري. هذا الدليل يلخص الحالة الحالية للمشروع وكيفية البدء.

## 🏗️ البنية التحتية والتقنيات
- **الواجهة الأمامية:** React 19 + Vite + Tailwind CSS 4.
- **قاعدة البيانات:** Supabase (مع تفعيل نظام RLS لعزل البيانات).
- **التشفير:** AES-256-GCM (أعلى معايير الأمان لحماية بيانات القضايا).
- **الذكاء الاصطناعي:** Gemini 1.5 Flash + Groq (لصياغة المستندات ومعالج واتساب).
- **التواصل:** Daily.co (غرف فيديو قانونية) + Meta WhatsApp Business API.

## 🌟 ميزات جاهزية المستثمرين (Investor Ready)
1. **العرض التفاعلي (One-Click Demo):** متاح في صفحة الدخول، يسمح للمستثمر باستكشاف النظام ببيانات واقعية (موكلين، قضايا، فواتير) بضغطة زر واحدة.
2. **تصميم Premium:** واجهة مستخدم احترافية تدعم الوضع الليلي، مع تحركات (Animations) سلسة.
3. **حوكمة متكاملة:** نظام شامل لإدارة القضايا المصرية بكافة درجات التقاضي (ابتدائي، استئناف، نقض).

## 🚀 خطوات النشر (Deployment)
المشروع جاهز للنشر التلقائي عبر ملفات الإعداد المرفقة:

### 1. النشر على Render (للسيرفر)
- اربط مستودع GitHub بحسابك في Render.com.
- اختر `Web Service`.
- سيقوم Render بالتعرف تلقائياً على ملف `render.yaml` وبناء الخدمة.
- تأكد من إضافة المفاتيح السرية (API Keys) في لوحة تحكم Render.

### 2. النشر على Vercel (للواجهة الأمامية)
- المشروع يحتوي على ملف `vercel.json` لإعدادات النشر السريع.
- استخدم زر `Deploy to Vercel` أو اربط المستودع مباشرة.

## 🔑 معلومات الدخول التجريبية
- **رابط الدخول:** `/login`
- **وضع المعاينة:** اضغط على زر **"بدء العرض التفاعلي للمنصة"** للدخول بصفحة "مدير مكتب" ببيانات كاملة.

## 📁 هيكلية الملفات الهامة
- `src/views/Landing.tsx`: صفحة الهبوط الرئيسية (براند مَلَف).
- `src/views/Dashboard.tsx`: لوحة القيادة الذكية.
- `routes/crypto.js`: محرك التشفير السيبراني.
- `services/ai/`: محرك الصياغة القانونية والذكاء الاصطناعي.
