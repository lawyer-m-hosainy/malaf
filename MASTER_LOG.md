# 📑 السجل العام للمشروع (MASTER LOG) - مَلَف (Malaf)

هذا الملف هو المرجع الرئيسي لكافة التعديلات، الإصلاحات، والتحسينات التي تمت على منصة "مَلَف" القانونية. يتم تحديثه دورياً لضمان توثيق مسار تطوير النظام.

---

## 📅 سجل اليوم: 14 مايو 2026 (تثبيت الأداء والواجهة)

### ✅ الإصلاحات الكبرى (Fixed)
- **استعادة الواجهة (UI Restoration):** تم إصلاح مشكلة حرجية تسببت في ظهور رسالة "HELLO WORLD" بدلاً من التطبيق، وتم استعادة نظام التوجيه (Routing) والموديولات بالكامل في `App.tsx`.
- **تثبيت البيانات (Data Persistence):** حل مشكلة "Unexpected Error" عند حفظ القضايا والفواتير عبر تنفيذ `Data Mapping` شامل بين الـ Frontend (camelCase) والـ Backend (snake_case).
- **منع التكرار (Duplicate Prevention):** إضافة نظام فحص ذكي يمنع تسجيل نفس الموكل مرتين بناءً على (الرقم القومي أو السجل التجاري).
- **إصلاح أخطاء التسجيل (React Error #185):** تم حل مشكلة (Maximum update depth exceeded) التي كانت تظهر بعد تسجيل الدخول بواسطة Google، عبر منع إعادة استدعاء أحداث المصادقة بشكل لا نهائي داخل الـ `AuthProvider`.
- **توافق الـ Schema:** إضافة طبقة تصفية في `legalDataService.ts` لحذف الحقول الزائدة قبل الإرسال لـ Supabase لمنع أخطاء (400 Bad Request).

### 🚀 التحسينات (Improved)
- **خيارات الموكلين:** إضافة صفات قانونية معقدة (مدعي ومدعى عليه فرعياً، متدخل، مستأنف، طاعن) مع مسميات ديناميكية للحقول.
- **اختيار المحاكم:** تحويل خانة "المحكمة/المكان" إلى حقل بحث ذكي يدعم الكتابة اليدوية الحرة لتغطية كافة محاكم الجمهورية.
- **تحديث الباقات والأسعار:** تم اعتماد الأسعار الجديدة للاشتراكات في الواجهة وقاعدة البيانات لتكون: الأساسية (300 ج.م)، المتقدمة (600 ج.م)، والمؤسسات (1300 ج.م) شهرياً.
- **جاهزية الإنتاج (Production Readiness):** إخفاء قسم "العرض التفاعلي" (Demo) وشريط التطوير من بيئة الإنتاج لضمان تجربة مستخدم احترافية تماماً.
- **المعايير التقنية (Accessibility):** إصلاح تحذيرات الوصول في مكون `CourtSelector` بإضافة الـ `title` المفقود.

## 🏗️ هيكلية النظام (Core Structure)

- **الواجهة الأمامية:** React (Vite) + Tailwind CSS + Lucide Icons.
- **قاعدة البيانات:** Supabase (PostgreSQL) مع تفعيل RLS.
- **الذكاء الاصطناعي:** Google Gemini API (الأساسي) + Groq (احتياطي).
- **الاستضافة:** Render (Web Service + Managed Database).
