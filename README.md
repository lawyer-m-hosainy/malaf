# ⚖️ مَلَف (Malaf) - منصة الإدارة القانونية الذكية المتكاملة

**مَلَف (Malaf)** هي منصة برمجية خدمية (SaaS) رائدة، صُممت خصيصاً لتمكين مكاتب المحاماة وشركات الاستشارات القانونية في جمهورية مصر العربية من التحول الرقمي الكامل. تدمج المنصة بين أحدث تقنيات الذكاء الاصطناعي وبين أدوات الإدارة القانونية الاحترافية لتقديم تجربة عمل سلسة، آمنة، ومنظمة.

---

## 🚀 الرؤية: رقمنة العدالة
نهدف في **مَلَف** إلى إنهاء عصر الإدارة الورقية العشوائية واستبدالها بلوحة تحكم رقمية ذكية. نحن نؤمن بأن وقت المحامي أثمن من أن يضيع في المهام الروتينية؛ لذا توفر المنصة الأتمتة الكاملة لإدارة القضايا، الموكلين، والشؤون المالية، مما يسمح للمحامين بالتركيز على الترافع الاستراتيجي وتطوير أعمالهم.

---

## ✨ المميزات الجوهرية (صُممت للمحامي المحترف)

### 1. 📂 إدارة القضايا والتقاضي الذكي (Litigation Management)
*   **ملف القضية الرقمي:** تنظيم شامل لكل قضية يشمل الخصوم، المحكمة، الدائرة، ونوع الدعوى.
*   **سجل الجلسات التفاعلي:** متابعة دقيقة لتواريخ الجلسات، القرارات، والطلبات المقدمة.
*   **التنفيذ القضائي:** نظام مخصص لمتابعة إجراءات التنفيذ، الحجوزات، والتحصيل المالي المرتبط بالأحكام.

### 2. �️ الأجندة القانونية والمواعيد (Smart Calendar)
*   **تنبيهات المواعيد:** نظام إشعارات ذكي يذكرك بمواعيد الجلسات، الطعون، والمواعيد القانونية الهامة.
*   **توزيع المهام:** إسناد المهام للمحامين المساعدين ومتابعة حالة التنفيذ بشكل فوري.

### 3. 🤖 المحامي الذكي (AI Legal Assistant)
يعمل بمحركات `Gemini` و `Groq` المتقدمة:
*   **صانع العقود الآلي:** توليد مسودات عقود (إيجار، عمل، تأسيس شركات) متوافقة مع القانون المصري بضغطة زر.
*   **محلل الوثائق:** تحليل مذكرات الخصوم واستخراج الثغرات القانونية ونقاط الضعف.
*   **المستشار القانوني الداخلي:** قاعدة معرفية قانونية للإجابة على التساؤلات الإجرائية والبحث في القوانين المصرية.

### 4. 💼 الإدارة المالية والضريبية (Legal Finance)
*   **الفاتورة الإلكترونية المصرية (ETA):** إصدار فواتير ضريبية متوافقة تماماً مع متطلبات مصلحة الضرائب المصرية (ضريبة 14%).
*   **حسابات الأمانات:** فصل كامل وشفاف بين أموال الموكلين (الأمانات) وأتعاب المكتب.
*   **تتبع المصروفات:** تسجيل الرسوم القضائية، الدمغات، والمصاريف الإدارية وربطها مباشرة بكل قضية.

### 5. �️ أمن البيانات والخصوصية (Bank-Grade Security)
*   **تشفير الطرف للطرف (E2EE):** يتم تشفير البيانات الحساسة (أرقام الهوية، المبالغ، الوثائق) محلياً في المتصفح باستخدام `AES-256` قبل إرسالها للسحابة.
*   **عزل البيانات (Multi-Tenancy):** استخدام سياسات `Row Level Security (RLS)` في `Supabase` لضمان استحالة وصول أي مكتب لبيانات مكتب آخر.
*   **سجل التدقيق (Audit Log):** تسجيل كل حركة دخول أو تعديل على البيانات لضمان أعلى معايير الشفافية داخل المكتب.

### 6. 🌐 بوابة الموكلين (Client Portal)
*   **تواصل احترافي:** واجهة مخصصة للموكلين لمتابعة قضاياهم، عرض الفواتير، وتحميل المستندات دون الحاجة للاتصال المتكرر بالمحامي.
*   **تحديثات تلقائية:** إرسال إشعارات للموكلين فور تحديث حالة الجلسة أو صدور قرار جديد.

## 🛠️ البناء التقني (Tech Stack)

*   **Frontend:** React 19, Vite, Tailwind CSS 4, shadcn/ui.
*   **Backend & Database:** Supabase (PostgreSQL), Edge Functions.
*   **AI Models:** Google Gemini Pro, Groq (Llama 3).
*   **Security:** Crypto-JS (AES-256), Supabase Auth.
*   **Payments:** Paymob Integration (Credit Cards, Wallets like Vodafone Cash).
*   **Communication:** Daily.co (Video Calls), WhatsApp Business API.
