# API Changelog — malaf.pro

سجل موحد لتتبع كافة التغييرات والتحسينات والإصلاحات الأمنية التي تطرأ على واجهات برمجة التطبيقات (API) لمنصة **ملف**.

### 💡 تنويه هام بخصوص الإصدارات
* يتم العمل حالياً على التجهيز لإطلاق الإصدار **v2** لدعم الترقيم المبني على المؤشر (Cursor-based Pagination) لتحسين كفاءة الاستعلام عن القوائم الكبيرة.

## [1.1.0] — 2026-05-26

### Added (إضافات جديدة)
* إضافة مسار جديد `/api/v1/cases/export` لتصدير القضايا إلى صيغة Excel/CSV.
* إضافة حقل اختياري `lawyer_notes` في استجابة تفاصيل القضية.

### Changed (تغييرات آمنة)
* تحسين زمن استجابة المسار `/api/v1/dashboard/metrics` بنسبة 40% من خلال تحسين فهارس الاستعلام (Query Indexes).

---

## [1.0.0] — 2026-01-15

### Added (إطلاق الإصدار الأول المستقر v1)
* إطلاق الإصدار الأول المستقر من الـ API بمسارات كاملة لإدارة:
  * **الموكلين (Clients)**: `/api/v1/clients`
  * **القضايا (Cases)**: `/api/v1/cases`
  * **المالية والمدفوعات (Finance)**: `/api/v1/finance`
  * **التقارير وسجلات التدقيق (Audit Logs)**: `/api/v1/audit`
* دعم ترويسة التحقق (Authorization: Bearer Token) عبر Supabase JWT.
* سياسات تأمين متكاملة لحماية البيانات وعزل الحسابات (RLS Policies).
