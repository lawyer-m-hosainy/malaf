# 🔍 Malaf.pro — E2E Audit Suite
## سكريبت التدقيق الشامل لمنصة ملف

> **وضع القراءة فقط — لا يُعدَّل أي كود أو بيانات في المنصة**

---

## 📦 التثبيت

```bash
# 1. انسخ هذا المجلد داخل مشروعك أو في مجلد منفصل
cd malaf-e2e-suite

# 2. ثبِّت المكتبات
npm install

# 3. ثبِّت متصفح الاختبار
npx playwright install chromium --with-deps
```

---

## 🚀 التشغيل

### تشغيل التدقيق الكامل (موصى به)
```bash
node run-audit.js
```

### تشغيل مجموعة اختبارات محددة
```bash
# الصفحة الرئيسية فقط
npx playwright test tests/01-homepage.spec.js

# التسجيل وتسجيل الدخول
npx playwright test tests/02-registration.spec.js

# وكلاء الذكاء الاصطناعي
npx playwright test tests/04-ai-agents.spec.js

# الدردشة والفيديو
npx playwright test tests/05-chat-video.spec.js
```

### تشغيل مع بيانات دخول حقيقية (لاختبار الصفحات المحمية)
```bash
MALAF_TEST_EMAIL=youroffice@email.com \
MALAF_TEST_PASSWORD=YourPassword123 \
node run-audit.js
```

### تشغيل مع المتصفح المرئي (Headed Mode) لمتابعة بصرية
```bash
npx playwright test --headed
```

## 📊 الاختبارات المضمنة

| الملف | ما يختبره |
|-------|-----------|
| `01-homepage.spec.js` | الصفحة الرئيسية، وقت التحميل، الروابط المكسورة، التجاوب مع الجوال |
| `02-registration.spec.js` | نموذج تسجيل المكتب، التحقق، تسجيل ببيانات وهمية |
| `03-dashboard.spec.js` | لوحة التحكم، الصفحات الداخلية، القائمة الجانبية |
| `04-ai-agents.spec.js` | **وكلاء الذكاء الاصطناعي** — جودة الردود، توافق الأسئلة، زمن الاستجابة |
| `05-chat-video.spec.js` | **دردشة النص ومكالمات الفيديو** — WebSocket، WebRTC، عناصر UI |
| `06-user-journey.spec.js` | رحلة المستخدم الكاملة، الأمان، Security Headers |

## 📁 التقارير المُنشأة

بعد التشغيل ستجد في مجلد `reports/`:

```
reports/
├── AUDIT_REPORT.txt       ← التقرير الرئيسي (عربي/إنجليزي)
├── full-audit.json        ← تقرير JSON تفصيلي
├── results.json           ← نتائج Playwright الخام
├── html-report/           ← تقرير HTML تفاعلي
│   └── index.html         ← افتحه في المتصفح
└── artifacts/
    └── screenshots/       ← لقطات شاشة لكل خطوة
```

لعرض التقرير التفاعلي:
```bash
npx playwright show-report reports/html-report
```

## 🤖 اختبار وكلاء الذكاء الاصطناعي

يختبر السكريبت الأسئلة التالية على الوكيل:

**أسئلة قانونية (يُتوقع الإجابة عليها):**
- إجراءات رفع دعوى مدنية
- تسجيل عقد إيجار
- حقوق إفلاس الشركة
- مدة التقادم التجاري
- رفع شكوى عمالية

**أسئلة خارج النطاق (يجب رفضها أو التحويل):**
- أسئلة عن المطاعم والطبخ

**حالات حافة:**
- حقل فارغ
- رموز فقط

**معايير التقييم:**
- ✅ الرد باللغة العربية
- ✅ يحتوي على مصطلحات قانونية
- ✅ طول مناسب
- ✅ لا رسائل خطأ تقنية
- ✅ معالجة صحيحة للأسئلة خارج النطاق

## 📹 اختبار الفيديو والشات

يفحص السكريبت:
- وجود عناصر UI (كتم الصوت، إيقاف الكاميرا، إنهاء المكالمة)
- دعم WebRTC في المنصة
- مكتبات الفيديو المُستخدمة (Agora, Twilio, Daily, Jitsi)
- اتصالات WebSocket للدردشة الحية
- أداء التحميل

---

## ⚠️ ملاحظات مهمة

1. **لا تعديلات**: الاختبارات تقرأ فقط — لا تُعدَّل أي ملفات أو قاعدة بيانات
2. **الحسابات الوهمية**: تُستخدم بيانات `mailinator.com` التي لا تؤثر على أي شيء حقيقي
3. **بيانات الدخول**: لأفضل نتيجة، مرر بيانات دخول حقيقية عبر متغيرات البيئة
4. **الصفحات المحمية**: بعض الاختبارات ستُحذَف تلقائياً إذا كانت الصفحة تتطلب مصادقة
