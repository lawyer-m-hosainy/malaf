## 🔲 مهام متبقية — مواقع خارجية (تُنجز لاحقاً)

### 4️⃣ إنشاء حساب Paymob (بوابة الدفع)
- [ ] اذهب إلى [accept.paymob.com](https://accept.paymob.com) وأنشئ حساباً جديداً
- [ ] أكمل التحقق من الهوية (KYC) — يحتاج بطاقة هوية + سجل تجاري
- [ ] بعد التفعيل، اذهب إلى **Dashboard → Settings → API Credentials** وانسخ:
  - [ ] `API Key` → أضفه في Render كـ `PAYMOB_API_KEY`
- [ ] اذهب إلى **Integrations** وأنشئ Integration جديد:
  - [ ] نوع **Card** (بطاقات ائتمان) → انسخ الـ ID → `PAYMOB_INTEGRATION_ID_CARD`
  - [ ] نوع **Mobile Wallet** (فودافون كاش) → انسخ الـ ID → `PAYMOB_INTEGRATION_ID_WALLET`
- [ ] اذهب إلى **iFrames** وأنشئ واحد → انسخ الـ ID → `PAYMOB_IFRAME_ID`
- [ ] اذهب إلى **Webhooks** واضبط:
  - [ ] Transaction Processed Callback: `https://malaf-platform.onrender.com/api/payment/callback`
  - [ ] انسخ الـ HMAC Secret → `PAYMOB_HMAC_SECRET`

### 5️⃣ إعداد WhatsApp Business API
- [ ] اذهب إلى [business.facebook.com](https://business.facebook.com) وأنشئ حساب Business
- [ ] اذهب إلى [developers.facebook.com](https://developers.facebook.com) وأنشئ **App** جديد (نوع: Business)
- [ ] فعّل منتج **WhatsApp** في الـ App
- [ ] في قسم **WhatsApp → Getting Started**:
  - [ ] اختر أو أضف رقم هاتف (سيكون رقم البوت)
  - [ ] انسخ **Temporary Access Token** (أو أنشئ Permanent Token)
- [ ] في قسم **WhatsApp → Configuration**:
  - [ ] Callback URL: `https://malaf-platform.onrender.com/api/whatsapp/webhook`
  - [ ] Verify Token: نفس القيمة في `WA_VERIFY_TOKEN` (اختر أي كلمة سرية)
  - [ ] اشترك في Events: `messages`
- [ ] أضف في Render Environment:
  - [ ] `WA_VERIFY_TOKEN` = الكلمة السرية التي اخترتها

### 6️⃣ إعداد صفحة فيسبوك + Messenger Bot
- [ ] أنشئ **صفحة فيسبوك** باسم "منصة ملف — إدارة القضايا" (أو أي اسم مناسب)
- [ ] اذهب إلى [developers.facebook.com](https://developers.facebook.com) → الـ App الذي أنشأته
- [ ] فعّل منتج **Messenger** في الـ App
- [ ] في قسم **Messenger → Settings**:
  - [ ] اربط الصفحة بالـ App
  - [ ] انسخ **Page Access Token** → أضفه في Render كـ `FB_PAGE_ACCESS_TOKEN`
- [ ] في قسم **Messenger → Webhooks**:
  - [ ] Callback URL: `https://malaf-platform.onrender.com/api/messenger/webhook`
  - [ ] Verify Token: `malaf_messenger_verify` (أو غيّرها وأضف `FB_VERIFY_TOKEN` في Render)
  - [ ] اشترك في Events: `messages`, `messaging_postbacks`
- [ ] انسخ **App Secret** من Settings → Basic → أضفه في Render كـ `FB_APP_SECRET`

### 7️⃣ إعداد مفاتيح الذكاء الاصطناعي (اختياري — للبوت الذكي)
- [ ] **Gemini**: اذهب إلى [aistudio.google.com](https://aistudio.google.com) → API Keys → أنشئ مفتاح → أضفه كـ `GEMINI_API_KEY`
- [ ] **Groq** (Fallback): اذهب إلى [console.groq.com](https://console.groq.com) → API Keys → أنشئ مفتاح → أضفه كـ `GROQ_API_KEY`
